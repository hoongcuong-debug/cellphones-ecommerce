# E-commerce Website – Mobile Phone Store

Website thương mại điện tử cho cửa hàng điện thoại di động: xem sản phẩm, tìm kiếm, xem chi tiết, giỏ hàng, checkout, đăng nhập/đăng ký.

## Tech Stack
- Frontend: React (Vite), TailwindCSS, React Router
- Backend: Node.js, Express
- Database: MySQL
- Others: Axios, REST API

## Main Features
- Product listing by category
- Product detail page
- Search products
- Cart (LocalStorage)
- Checkout (demo)
- Login / Register

## Run Project
## Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
npm run dev
